## Overview

This repository documents a complete JavaScript learning journey.  
It starts from fundamentals and moves into deep internals and real-world engineering.

This guide is for:

- Developers preparing for interviews
- Professionals aiming for senior roles
- Engineers who want deep internal understanding of JavaScript

## Phase 1: Foundations

### Variables & Scope

JavaScript stores values using variables:

- `var` (function-scoped, legacy)
- `let` (block-scoped)
- `const` (block-scoped, reference cannot be reassigned)

Concepts covered:

- Scope (global, function, block)
- Hoisting behavior
- Reassignment rules
- Temporal Dead Zone (TDZ)
- Best practices

---

### Data Types

Primitive types:

- String
- Number
- Boolean
- Null
- Undefined
- Symbol
- BigInt

Non-primitive:

- Object (includes arrays and functions)

Key concepts:

- Type coercion
- `typeof` behavior (including common gotchas)
- Value vs reference
- Deep vs shallow copy

---

### Operators

Types:

- Arithmetic
- Comparison
- Logical
- Assignment
- Ternary
- Nullish coalescing (`??`)
- Optional chaining (`?.`)

Understanding:

- Operator precedence
- Short-circuit evaluation
- Strict vs loose equality (`===` vs `==`)

---

### Conditional Statements

Decision making:

- `if`, `else if`, `else`
- `switch`

Concepts:

- Truthy and falsy values
- Guard clauses
- Early returns
